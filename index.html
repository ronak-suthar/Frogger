Game Information and Instructions

In this game, the frog wants to reach the endpoint which is the stone on another end.

  • But there are obstacles in the way.

  • We have to clear two obstacles road and river

  • Valid Ways to Clear the Obstacles
    • To cross the Road we must not hit a car.
    • To cross the River we must move upon the logs only.
    • Use the Keyboard Arrow Keys to Move Frog Up, Left, Down, and Right.

  • The Current Time Remaining and Result can be viewed on The Green Status Bar.

  • Press The Start Button To Start Or Pause The Game

  • So let's Go Hit the Start Button get the fingers on Arrow Keys!

This Game is Requires KeyBoard and Screen Width of Min 700px.
