Frogger

Time Left : 20

Result : Please Start The Game

In this game the frog wants to reach the end point which is the stone on other end.

  • But there are obstacles in the way.

  • We have to clear two obstacles road and river

  • Valid Ways to Clear the Obstacles
    • To cross the Road we must not hit a car.
    • To cross the river we must move upons the logs only.
    • Use the Keyboard Arrow Keys to Move Frog Up, Left, Down and Right.

  • The Current Left Time and Result is Displayed on The Green Status Bar

  • Press The Start Button To Start Or Pause The Game

  • So lets Go Hit the Start Button get the fingers on Arrow Keys !

This Game is Requires KeyBroad and Screen Width of Min 750px.
